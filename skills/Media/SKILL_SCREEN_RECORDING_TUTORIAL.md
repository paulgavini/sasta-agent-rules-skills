# Skill: Screen Recording Tutorial

## Purpose

This skill allows the agent to plan scripts, shot lists and support notes for clear, accessible and privacy-safe screen recording tutorials.

## When To Use

Use this skill when the user asks for:

- a screen recording script
- tutorial narration
- step-by-step recording notes
- an LMS video guide
- a software walkthrough
- a student or staff explainer video

## Inputs

- software, website or platform being recorded
- audience and prior knowledge
- task goal and expected end state
- account, login or privacy constraints
- recording length target
- whether captions, transcript or audio description are needed
- output destination

## Process

1. Define the one task the recording teaches.
2. Remove unnecessary setup, clutter and private information from the screen.
3. Script the opening, steps and closing call to action.
4. Use short, observable actions: click, type, choose, save, submit.
5. State what the viewer should see after each key step.
6. Pause for loading or transitions only when learners need that cue.
7. Include caption and transcript needs.
8. Avoid showing passwords, student data, emails, marks or private messages.
9. Keep troubleshooting notes separate from the main recording unless essential.

## Output

```markdown
# Screen Recording Plan

## Goal
## Before Recording
## Script
## On-Screen Actions
## Accessibility Notes
## Privacy Checks
## Transcript Notes
```

## Quality Criteria

- The tutorial teaches one clear workflow.
- The screen is privacy-safe.
- Steps are observable and concise.
- Captions or transcript needs are planned.
- The script matches what appears on screen.
- The recording is short enough for the audience.
