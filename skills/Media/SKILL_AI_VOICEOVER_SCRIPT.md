# Skill: AI Voiceover Script

## Purpose

This skill allows the agent to create voiceover scripts for AI or human narration that are clear, ethical, accessible and suitable for school audiences.

## When To Use

Use this skill when the user asks for:

- a voiceover script
- narration for a video, explainer or tutorial
- AI-generated voice guidance
- timing notes for audio
- a script to accompany slides or screen recording
- student-facing spoken instructions

## Inputs

- audience and year level
- video or slide purpose
- desired tone and length
- whether AI voice generation will be used
- pronunciation needs for names, terms or acronyms
- privacy, consent, copyright or school-policy constraints
- required transcript or captions

## Process

1. Define the purpose and audience.
2. Write for listening, using short sentences and natural phrasing.
3. Mark pauses, emphasis and pronunciation only where useful.
4. Keep one idea per sentence where possible.
5. Avoid jokes, idioms or references that may not work for all learners.
6. Include transcript and caption alignment notes.
7. Do not imitate a real person's voice without consent.
8. Avoid using personal student information in prompts or scripts.
9. Label AI-generated narration where transparency is required by school policy.

## Output

```markdown
# Voiceover Script

## Context
## Script
## Timing Notes
## Pronunciation Notes
## Accessibility And Consent Checks
```

## Quality Criteria

- The script sounds natural when read aloud.
- Sentences are concise and clear.
- Timing is realistic.
- Accessibility needs are considered.
- Consent and privacy risks are identified.
- AI use is transparent where required.
