# Rule: Australian Language

## Purpose

Ensure the agent writes in Australian English and uses language that suits Australian school settings. This rule keeps spelling, terminology, tone and examples consistent for South Australian secondary teaching work.

## Applies To

- Student-facing resources, tutorials, worksheets and assessment tasks
- Teacher notes, lesson plans, marking guides and rubrics
- Parent, caregiver, staff and leadership communication
- Word, LMS, presentation and export-ready documents
- Curriculum, SACE and school-policy references

## Rule Statement

The agent must use Australian spelling, Australian school terminology and respectful, inclusive language unless the user explicitly requests another style for a specific audience or publication context.

## Requirements

- Use Australian English spelling, such as `behaviour`, `analyse`, `organisation`, `modelling`, `focussed` or the user/school-preferred variant where supplied.
- Check uncertain spellings against an Australian English dictionary or official Australian style source.
- Use Australian school terms such as `Year 8`, `lesson`, `learning intention`, `success criteria`, `assessment task`, `rubric`, `parent` or `caregiver` as appropriate.
- Refer to South Australian and SACE contexts accurately when they are relevant.
- Use plain language that is clear for the intended audience.
- Use inclusive, respectful language for students, families, staff, Aboriginal and Torres Strait Islander peoples, disability, neurodiversity, culture, gender and age.
- Preserve technical terms where they are needed for Science, Digital Technologies, IT, Electronics, Robotics or assessment.
- Avoid US school terms unless the user requests them or they are part of a software interface.

## Decision Logic

Apply this rule to every generated file or response. Use Australian English by default. If the user supplies a school style preference, follow it unless it conflicts with accuracy, safety or respectful language. If software documentation uses US interface labels, keep the exact interface label but explain in Australian English around it.

## Examples

- Write `Year 9 students will analyse the data`, not `Grade 9 students will analyze the data`.
- Use `behaviour expectations` in a relief lesson note, not `behavior expectations`.
- Keep a software menu label such as `Color` if the platform displays it that way, but write surrounding instructions in Australian English.
- Use `Aboriginal and Torres Strait Islander peoples` when referring broadly to First Nations peoples in Australia.

## Non-Examples

- Using `semesterized`, `sophomore`, `middle schooler` or `grading period` for an Australian school resource.
- Rewriting a student-facing worksheet with unnecessary jargon.
- Using labels for students or families that are deficit-based, dismissive or culturally insensitive.

## Quality Check

- Is the spelling Australian English?
- Are school terms appropriate for South Australia and the year level?
- Is the tone respectful, inclusive and audience-appropriate?
- Are software labels preserved where exact labels matter?
- Are technical terms used only where they help accuracy or clarity?
