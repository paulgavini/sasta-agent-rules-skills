# Skill: Data Analysis

## Purpose

This skill allows the agent to create classroom data-analysis tasks that move from a question, through clean data and representation, to evidence-based conclusions.

## When To Use

Use this skill when the user asks for:

- a data analysis lesson
- spreadsheet or dashboard data tasks
- graph, chart or table interpretation
- CSV or survey analysis
- sample datasets
- conclusions from data

## Inputs

- learning question or investigation question
- dataset or data source
- data types: categorical, numerical, text, image, sound or relational
- tool: Excel, Power BI, Python, database, manual table or other
- year level and mathematical readiness
- privacy and sensitivity level
- required chart, table or summary

## Process

1. Define the question the data should answer.
2. Check whether the data is real, sample, synthetic or student-generated.
3. Identify data types, fields, units and missing values.
4. Clean only what is necessary and document changes.
5. Choose a representation that fits the question.
6. Calculate simple summaries where useful.
7. Interpret patterns, outliers and limitations.
8. Avoid claiming causation unless the data supports it.
9. Protect personal data through minimisation, aggregation or anonymisation.

## Output

```markdown
# Data Analysis Task

## Question
## Dataset
## Cleaning
## Representation
## Findings
## Limitations
## Privacy Checks
```

## Quality Criteria

- The data matches the question.
- Chart choice suits the data type.
- Units and labels are clear.
- Limitations are stated.
- Privacy is protected.
- Conclusions stay within the evidence.
