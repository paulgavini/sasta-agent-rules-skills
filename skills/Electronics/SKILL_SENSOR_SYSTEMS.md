# Skill: Sensor Systems

## Purpose

This skill allows the agent to create sensor-system activities where students collect, interpret and use data from digital or electronic sensors.

## When To Use

Use this skill when the user asks for:

- sensor lessons
- micro:bit sensor activities
- data logging tasks
- input-processing-output systems
- environmental monitoring tasks
- sensor troubleshooting
- electronics or robotics projects involving measured data

## Inputs

- sensor type and platform
- quantity being measured
- sampling frequency or observation interval
- output or response required
- available display, storage or data export method
- calibration or comparison method if needed
- safety, privacy and environmental constraints

## Process

1. Define what the sensor measures and why.
2. Identify units, range and expected values.
3. Test raw sensor readings before using them in decisions.
4. Decide whether readings need smoothing, thresholds or calibration.
5. Record data in a clear table if analysis is required.
6. Connect sensor input to output behaviour only after readings are understood.
7. Check for environmental factors that affect readings.
8. Use safe, low-risk data and avoid collecting personal information unless approved.
9. Interpret results with limitations.

## Output

```markdown
# Sensor System Activity

## Purpose
## Sensor And Measurement
## Setup
## Data Collection
## Processing Or Thresholds
## Output Behaviour
## Limitations
## Safety And Privacy
```

## Quality Criteria

- The sensor and measured quantity are clear.
- Units, range and limitations are considered.
- Raw data is checked before decisions are made.
- Output behaviour follows from input data.
- Privacy and safety are handled.
- Conclusions do not exceed the data.
