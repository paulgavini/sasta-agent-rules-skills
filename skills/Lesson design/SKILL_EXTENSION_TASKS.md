# Skill: Extension Tasks

## Purpose

This skill allows the agent to create extension tasks that deepen learning, increase transfer or add complexity without turning optional challenge into hidden required work.

## When To Use

Use this skill when the user asks for:

- extension activities
- challenge tasks
- early finisher work
- enrichment prompts
- optional higher-order questions
- deeper application tasks
- differentiated extension for lessons, tutorials or assessments

## Inputs

- year level or SACE stage
- subject, topic and core task
- required learning goal
- expected prior knowledge
- available time and resources
- whether the extension is optional, formative or assessed
- curriculum, assessment or rubric constraints
- student audience and platform

## Process

1. Identify the core required task and keep it separate from extension.
2. Decide the purpose of extension: deeper explanation, transfer, evaluation, design, comparison, error analysis or independent inquiry.
3. Increase cognitive demand through reasoning or application, not just by adding more questions.
4. Keep extension aligned to taught or accessible knowledge unless the user asks for independent research.
5. Label extension clearly as optional, challenge or extension unless it is formally assessed.
6. Avoid introducing advanced concepts that undermine year-level appropriateness.
7. Include enough scaffolding for students to start without giving away the core answer.
8. For assessment contexts, avoid adding hidden criteria or unfair advantage.
9. Keep the extension realistic for early finishers or the stated lesson time.
10. Add teacher notes when an extension changes evidence, equipment, safety or marking load.

## Output

```markdown
# Extension Tasks

## Extension Purpose

## Optional Challenge

## Deeper Thinking

## Transfer Or Application

## Teacher Notes
```

Adjust headings to the user's requested output and task type.

## Quality Criteria

- Extension is clearly separate from required work.
- Challenge comes from deeper thinking, not more volume.
- Tasks remain age-appropriate and aligned.
- Students have a clear starting point.
- Assessment fairness is protected.
- Teacher notes identify any extra resources or marking implications.

## Example User Requests

- `Add extension tasks to this Year 8 Science worksheet.`
- `Create early finisher challenges for a Scratch lesson.`
- `Write optional challenge questions for this graphing task.`
- `Add extension without making the assessment harder.`

## Failure Modes

- Making extension required without saying so.
- Adding unrelated busywork.
- Introducing senior content into a junior lesson without need.
- Rewarding speed rather than depth.
- Creating tasks that need resources students do not have.
- Adding hidden assessment expectations.

## Completion Checklist

- Is extension visibly optional or clearly assessed?
- Does it deepen the same learning goal?
- Is it age-appropriate?
- Is it realistic for the available time?
- Are extra resources or marking implications clear?
- Does it avoid unfair penalty or hidden criteria?
