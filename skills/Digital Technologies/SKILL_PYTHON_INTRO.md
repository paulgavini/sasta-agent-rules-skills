# Skill: Python Intro

## Purpose

This skill allows the agent to create beginner Python resources that teach programming concepts through small, readable and testable examples.

## When To Use

Use this skill when the user asks for:

- an introductory Python lesson
- a beginner Python worksheet
- code examples for variables, input, output, selection, iteration or functions
- Python debugging support
- Python answer keys or worked examples
- student-facing Python tutorials

## Inputs

- year level and prior programming experience
- coding environment: browser editor, local Python, Thonny, Replit, notebook or other
- target concept
- expected input and output
- whether code must be copied, traced or modified
- assessment conditions
- device or installation constraints

## Process

1. Confirm the learner's current level and coding environment.
2. Start with the problem and expected behaviour.
3. Use one new Python idea at a time.
4. Write short, readable code using clear names and standard indentation.
5. Show expected output separately.
6. Add a "try this change" step before open-ended creation.
7. Include common error checks, especially indentation, spelling and data type issues.
8. Avoid advanced features unless requested or clearly labelled as extension.
9. State whether code has been tested or is illustrative.

## Output

```markdown
# Python Activity

## Goal
## Starter Code
## What It Does
## Your Turn
## Expected Output
## Debugging Checks
```

## Quality Criteria

- Code is readable and copyable.
- The concept is not buried under extra syntax.
- Indentation is preserved.
- Inputs and outputs are clear.
- Debugging support is specific.
- The task fits the stated environment.
