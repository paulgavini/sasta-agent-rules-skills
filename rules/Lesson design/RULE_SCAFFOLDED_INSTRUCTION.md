# Rule: Scaffolded Instruction

## Purpose

Ensure lesson materials support students to move from teacher guidance to independent practice without overwhelming working memory or hiding essential steps.

## Applies To

- student tutorials and worksheets
- lesson sequences
- worked examples
- practical investigations
- coding and digital technology tasks
- assessment preparation
- relief lessons

## Rule Statement

The agent must provide scaffolding that matches the students' prior knowledge, task complexity and intended independence.

## Requirements

- Start from what students are likely to know.
- Break complex tasks into clear stages.
- Model the process before expecting independent application where needed.
- Use worked examples, sentence starters, templates or checklists only when they support learning.
- Fade scaffolds for more advanced students or later practice.
- Keep each step purposeful and tied to the learning goal.
- Avoid adding so many prompts that students only follow instructions without thinking.
- Use checks for understanding before independent work where relevant.
- Provide support for common barriers such as vocabulary, data tables, code structure or graphing conventions.

## Decision Logic

Use heavier scaffolding for new concepts, junior students, complex tasks, relief lessons or mixed-ability classes. Use lighter scaffolding when students need independent evidence for assessment. If the task is an assessment, ensure scaffolding does not give undue assistance.

## Examples

- A practical report scaffold gives headings and prompts for variables, method, data, conclusion and evaluation.
- A coding tutorial models one example, then gives a similar guided task, then an independent challenge.
- A graphing lesson gives labelled axes in the first example and asks students to label their own in the next task.

## Non-Examples

- Giving Year 8 students a blank senior report structure with no guidance.
- Providing a complete answer pattern in a summative assessment.
- Adding a long checklist that distracts from the core learning.
- Using scaffolds that do the thinking for students.

## Quality Check

- Does the scaffold match the task difficulty?
- Are steps clear and purposeful?
- Is student thinking still required?
- Are supports faded where appropriate?
- Is assessment integrity protected?
- Are common barriers addressed?
