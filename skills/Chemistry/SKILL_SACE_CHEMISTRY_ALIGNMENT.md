# Skill: SACE Chemistry Alignment

## Purpose

This skill allows the agent to align Chemistry resources, lessons and assessment tasks to SACE Chemistry using current SACE sources or supplied school material.

## When To Use

Use this skill when the user asks for:

- SACE Chemistry alignment
- Stage 1 or Stage 2 Chemistry task support
- investigations folio guidance
- skills and applications task design
- LAP or subject outline alignment
- chemistry assessment evidence mapping
- SACE Chemistry teacher notes

## Inputs

- SACE stage and assessment type
- task, lesson or resource to align
- current SACE Chemistry subject outline or supplied school material if available
- LAP, assessment conditions or school requirements
- topic, evidence and student output
- performance standards or criteria supplied by the user
- moderation, verification or academic integrity requirements

## Process

1. Confirm Stage 1 or Stage 2 Chemistry.
2. Check current SACE Chemistry pages or supplied subject-outline/LAP material before making specific claims.
3. Treat the current supplied or official Chemistry subject outline as the primary authority for Chemistry-specific content, assessment types, weightings, criteria and performance standards; use general SACE policy sources for LAP, integrity, moderation and verification processes.
4. Identify the assessment type, such as investigations folio or skills and applications task.
5. Map the resource to the evidence students produce.
6. Keep assessment conditions, verification and academic integrity requirements visible.
7. Use current SACE terminology only when supplied or verified.
8. Avoid inventing performance-standard wording, weightings or task rules.
9. Flag when the school LAP or current subject outline must be checked.
10. Separate student-facing instructions from teacher-facing alignment notes.
11. Keep the output useful for planning, moderation and classroom implementation.

## Output

```markdown
# SACE Chemistry Alignment

## Stage And Assessment Type

## Source Checked

## Intended Evidence

## Alignment Notes

## Assessment Integrity Notes

## LAP Or Local Checks
```

Use the user's supplied template when provided.

## Quality Criteria

- Stage and assessment type are clear.
- SACE claims are sourced or flagged for checking.
- Evidence aligns with the task.
- Integrity and verification are considered.
- LAP and local checks are visible.
- Unsupported SACE wording is avoided.

## Example User Requests

- `Align this Stage 2 Chemistry practical to SACE.`
- `Check whether this task fits a Chemistry investigations folio.`
- `Write SACE Chemistry teacher notes for this assessment.`
- `Map this reaction rates lesson to SACE Chemistry evidence.`

## Failure Modes

- Inventing SACE requirements from memory.
- Mixing Stage 1 and Stage 2 expectations.
- Claiming LAP compliance without checking the LAP.
- Hiding assessment integrity conditions.
- Using generic chemistry alignment when SACE-specific alignment was requested.
- Adding performance-standard wording not supplied or verified.

## Completion Checklist

- Is the SACE stage clear?
- Has a current SACE or supplied source been checked?
- Is assessment type identified?
- Is required evidence clear?
- Are integrity and local checks visible?
- Are unverifiable claims avoided?
