# Rule: Variables And Fair Testing

## Purpose

Ensure practical investigations and inquiry tasks handle variables, controls and fair testing accurately for the intended year level.

## Applies To

- practical investigation planning
- student worksheets and report scaffolds
- data collection tables and graphing tasks
- assessment tasks and rubrics involving inquiry
- chemistry, physics, biology and junior science activities

## Rule Statement

The agent must clearly identify independent, dependent and controlled variables where relevant, and must design fair tests that can generate evidence for the question being investigated.

## Requirements

- State the investigation question before identifying variables.
- Identify the independent variable as what is changed or compared.
- Identify the dependent variable as what is measured or observed.
- Identify controlled variables as factors kept the same to make the test fair.
- Explain variables in age-appropriate language for junior students.
- Avoid fake fair tests where multiple important variables change at once.
- Include repeated trials or sufficient data where appropriate.
- Match the data table to the variables and planned measurements.
- Recognise when an investigation is not a fair test, such as surveys, modelling, classification or secondary-data analysis, and use more suitable language.
- Ensure conclusions refer back to the investigated relationship.

## Decision Logic

If the activity tests the effect of one factor on another, use fair-test language. If the activity explores, models, classifies or analyses secondary data, avoid forcing independent/dependent/control labels where they do not fit. If a student task asks for variables, make them visible and check that the method actually controls them.

## Examples

- In testing how ramp height affects cart speed, ramp height is the independent variable and speed is the dependent variable.
- A data table should include units and columns that match the measured dependent variable.
- In a survey about study habits, use "factors compared" and "data collected" if fair-test language is misleading.
- A method should keep equipment, timing and measurement technique consistent where they affect results.

## Non-Examples

- Calling every investigation a fair test.
- Listing controlled variables that the method does not actually control.
- Changing both temperature and concentration while claiming to test only temperature.
- Asking for a conclusion that does not answer the investigation question.
- Using senior terminology in a junior worksheet without explanation.

## Quality Check

- Is the investigation question clear?
- Are variables correctly identified?
- Does the method control important factors?
- Is the data table aligned to the variables?
- Are repeated trials or sufficient data considered?
- Does the conclusion answer the question?
